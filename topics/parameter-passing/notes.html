1.2. parameter passing - example EOPL page 130

(let ([p (lambda (x) (set! x 1))]) (let ([a 3]) (begin (p a) a))) ===> 3 (in Scheme)

  • call-by-reference ===> 1

(let ([p (lambda (x) (let ([x 1]) x))]) (let ([a 3]) (begin (p a) a))) ===> 3 (in Scheme & call-by-reference)

1.3. encapsulation

  • lexical scope - controls visibility of bindings
  • encapsulation - control the mutation of variables
    • as a side-effect of functions
    • change only through function not through any other means
  • example

#lang racket (define ms (let ([x 5]) (let ([getx (lambda () x)] [setx (lambda (v) (set! x v))]) (list getx setx))))

(define getx (first ms)) (define setx (second ms)) (getx) ===> 5 (setx 7) (getx) ===> 7 ;x ===> x undefined
