Principles of Programming Languages: Syllabus

1. Syllabus

Abstract Reduction Systems:
termination, normal forms, confluence, local confluence, Newmann's Lemma.
Language principles:
Back-Naur Form. Abstract syntax trees. Scope and lexical environments. Scope vs. Extent. `Dynamic' Scope. Stores: first class references. Expressible, Denotable and Storable value domains. Aggregates and Parameter passing: call-by-value, call-by-name, call-by-need. Runtime contexts. Recursion and tail recursion. Recursion and the Y combinator.
Lambda calculus:
Syntax, Substitutions and alpha equivalence, beta and eta reductions. De Bruijn notation. Confluence of lambda calculus: Church-Rosser theorem. Reduction strategies: normal and applicative order reduction.
Continuations and Control abstraction:
Runtime contexts, continuations, Continuation passing style and continuation semantics. Exceptions and threads. Delimited continuations.
Types:
Type safety. Preservation and progress theorems. Type checking and type inference. Simply-typed lambda calculus. Term unification. Hindley-Milner polymorphism. Milner's W algorithm. Monads.

(Varying subsets of the last section on Types are covered in the course based on time constraints.)

2. Assignments

Throughout the course, operational semantics will be employed to formalise the understanding of language principles. Students assignments will include implementation of interpreters implementing many of the language features above. Interpreters written in a functional programming language like Racket, Haskell or OCAML.
