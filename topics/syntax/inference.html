1. Introduction

1.1. Inference

Inference
We study the process of inference in logic.
Use
inference (also called deduction) is a way of reasoning. It is used in all mathematics and much of computer science.

1.2. Judgements

Judgement
A judgement is a proposition. For example, we would like to make the judgement 3+4 is an expression. Or that 15th Aug 2020 is a Thursday.
Inference
In reasoning, we want to connect judgements. A set of judgements may lead to another judgement. E.g., from the judgements
+ It is raining

+ It is not sunny when it rains

I can deduce that 

It is not sunny

Judgements are of the form \[ e\ Exp\]

The rules of inference determine a logic. Sound judgements are those that can be derived using the rules of inference.

Each rule has the form 

 <antecedent> ... 
--------------------  (side condition)       NAME
 <consequent>
  • <antecedent> … and denotes a set of judgements
  • <consequent> is a judgement
  • (side condition) is any condition that is `extra-logical'
  • NAME a name of the rule.
